## About The Project


There are scripts out there that help you to collect financial data. These places include yahoo finance, bloomberg as well as many other financial websites. However, I realised that many of these websites contain so many errors that it is best to look at the raw files yourself. This script pulls data from the SEC 10-k filings, which are as accurate as you can possibly get.

This script has been written with Python





<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.

  ```sh
  Download the requirements.txt file listed in this repository.
  
  CD to the directory that requirements.txt is located and run:
  pip install -r requirements.txt
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/BrandonLow96/Scraping-Code.git
   ```
2. Install packages
   ```sh
   pip install -r requirements.txt
   ```
